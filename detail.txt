If your API key ever expires, replacing it is extremely simple. You have two places where you can easily swap it out:

Method 1: Using the Environment File (Recommended)

In your code editor, open the 

.env
 file located at the very root of your Flight_Tracker folder.
You will see this line:
env
VITE_AVIATIONSTACK_KEY=78ec915ed4be71a26ecdc6b6b9694bca
Simply replace that string with your new API key!
Method 2: Directly in the API Code

Open up 

src/api/flightApi.js
.
Scroll to Line 24 inside the 

fetchFlightSchedules
 function.
You will see it structured like this as a fallback:
javascript
const apiKey = import.meta.env.VITE_AVIATIONSTACK_KEY || '78ec915ed4be71a26ecdc6b6b9694bca';
Just delete the old key inside the quotation marks and paste your new one there.
Once you replace it in either spot, completely restart your Vite terminal server (npm run dev) just to make sure it loads the new key, and you'll be back fully functional again!
